# Deskripsi mini project
1. Mengurutkan nilai siswa dari yg tertinggi ke terendah

3. menampilkan siswa yg nilainya diatas rata-rat
4. menampilkan tingkatan tau grade untuk setiap siswa

# stack yang saya gunakan

# software pendukung

text editor --> visual studio code

2. klik fitur Go Live di vsc untuk menuju browser
3. di broser klik kana pilih inspect
4. lalu klik console
